<h1 align="center">GamerTech Connect</h1>

<p align="center">
  Remote desktop support tool for <a href="https://gamertech.ca/">GamerTech Gaming PC</a> customers.
</p>

---

GamerTech Connect is a custom remote support application built on [RustDesk](https://github.com/rustdesk/rustdesk), providing easy and secure remote assistance for Windows desktops.

## Features

- Peer-to-peer connections with relay fallback
- Screen sharing, file transfer, and remote input
- Low latency with hardware codec support
- Simple interface for both customers and technicians
